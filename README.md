# Neo CRM System

A full-stack CRM (Customer Relationship Management) application built using the MERN stack.

## Features
- Admin login system
- Add, view, update, and delete leads
- Lead status tracking (new, contacted, converted)
- Notes and follow-ups
- Search/filter leads
- Analytics dashboard with charts
- Timestamp tracking

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend/vite-project
npm install
npm run dev
